CAPAS

SUELO CAMINABLE

Clic en el lienzo = añadir punto de la curva. Clic derecho = quitar el más cercano. Verde. Si no hay puntos, el suelo es plano.

ZONAS

Tipo a dibujar (arrastra en zona vacía para crear). Clic en una zona = seleccionar; arrastra su cuerpo para mover o las esquinas para cambiar el tamaño. Supr la borra.

✨ CREAR CON IA (Nano Banana)

Genera un fondo o un elemento en el estilo del juego (ukiyo-e). Tarda ~15-30 s. Requiere el server local y la API key.
